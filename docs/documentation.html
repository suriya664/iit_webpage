ACyberCloud Template Documentation

Getting Started

This template is ready to use. Simply upload all files to your web server and start customizing.

File Structure


acybercloud-template/
├── assets/
│   ├── css/        # Stylesheets
│   ├── js/         # JavaScript files
│   ├── img/        # Images
│   └── fonts/      # Font files
├── pages/          # HTML pages
├── partials/       # Reusable components
└── docs/          # Documentation

Customization Guide

Colors

Edit CSS variables in assets/css/style.css:


:root {
    --primary-color: #0466c8;
    --secondary-color: #0a192f;
    --accent-color: #00b4d8;
}

Content

Update content in HTML files located in the pages/ directory.

Images

Replace placeholder images in assets/img/ with your own images.
